# gofetch

System information tool written in Golang 

# Available OS

GNU/Linux & Android

# Dependencies

```
lsb-release, xrandr
```

# Install

```bash
git clone https://github.com/rendick/gofetch.git
cd gofetch
go build
./gofetch
```
